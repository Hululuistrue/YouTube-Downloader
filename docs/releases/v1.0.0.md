# Release v1.0.0

Release date: 2026-03-01
Version: `1.0.0`

## Summary

First public MVP release of the web-based YouTube downloader, including full async task pipeline, cookie upload and management, exact format selection, staged progress display, cancellation support, and bilingual web UI.

## Highlights

- Real download pipeline powered by `yt-dlp` + `ffmpeg`.
- Async processing with API + Redis queue + worker.
- Persistent task state and history with PostgreSQL.
- Cookie file upload/list/delete and task binding (`cookieFileId`).
- Parse API returns real available formats from `yt-dlp`.
- Exact format download by `formatId`.
- Retry and cancel task APIs.
- Staged progress model (`queued`, `downloading`, `transcoding`, `uploading`, `success/failed`).
- Signed temporary download URL (`/api/v1/download/:taskId?token=...`).
- Bilingual pages: English default (`/`), Chinese (`/index.html`).

## Included API Endpoints

- `GET /health`
- `POST /api/v1/parse`
- `GET /api/v1/cookies`
- `POST /api/v1/cookies`
- `DELETE /api/v1/cookies/:cookieFileId`
- `POST /api/v1/tasks`
- `GET /api/v1/tasks`
- `GET /api/v1/tasks/:taskId`
- `POST /api/v1/tasks/:taskId/retry`
- `POST /api/v1/tasks/:taskId/cancel`
- `GET /api/v1/download/:taskId?token=...`
- `POST /api/v1/complaints`

## Operational Notes

- Requires PostgreSQL, Redis, `yt-dlp`, and `ffmpeg`.
- Browser-cookie mode is disabled by default to avoid DPAPI issues on Windows.
- For protected YouTube videos, upload a fresh user cookie file (`cookies.txt`) before parse/download.
- Anonymous users are rate/concurrency limited.

# 发布说明 v1.0.0（中文）

发布时间：2026-03-01  
版本：`1.0.0`

## 概览

这是本项目首个公开 MVP 版本，已包含完整异步下载链路、Cookie 上传管理、格式精确选择、阶段进度展示、任务取消能力以及中英文双语页面。

## 重点能力

- 基于 `yt-dlp` + `ffmpeg` 的真实下载与转码。
- API + Redis 队列 + Worker 的异步任务架构。
- PostgreSQL 持久化任务状态与事件。
- Cookie 文件上传/列表/删除，并可绑定到下载任务（`cookieFileId`）。
- 解析接口返回真实可用音视频格式。
- 支持按 `formatId` 精确下载指定格式。
- 支持任务重试与取消。
- 支持阶段化进度（排队/下载/转码/落盘/完成或失败）。
- 支持签名下载链接（默认临时有效）。
- 默认英文页面（`/`）+ 中文页面（`/index.html`）。

## 运行依赖

- PostgreSQL
- Redis
- `yt-dlp`
- `ffmpeg`

## 使用建议

- 对受限视频，请先上传最新且有效的用户 Cookie 文件后再解析下载。
- Windows 环境默认关闭浏览器直接取 Cookie 的模式，以减少 DPAPI 相关失败。
- 匿名用户会受到限额与并发限制。
